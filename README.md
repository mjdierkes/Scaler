# Scaler

A SwiftUI-based iOS application.

## Requirements

- iOS 17.0+
- Xcode 15.0+
- XcodeGen

## Setup

1. Install XcodeGen if you haven't already:
```bash
brew install xcodegen
```

2. Generate the Xcode project:
```bash
xcodegen generate
```

3. Open the generated Xcode project:
```bash
open Scaler.xcodeproj
```
